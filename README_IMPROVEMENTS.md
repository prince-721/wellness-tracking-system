## 🚀 How to Use New Features

### **Reminders Setup** (3 minutes)
1. Go to Profile → "Manage Reminders"
2. Click "+ Add Reminder"
3. Set time, type, and frequency
4. Done! You'll get notified automatically

### **Build Favorite Meals** (Automatic)
1. Log meals normally via Calories screen
2. Frequently logged meals auto-become favorites
3. Go to Profile → "Favorite Meals" to see them
4. Click "➕ Log Meal" for instant logging next time

### **Track Streaks** (Automatic)
1. Home screen shows your current streaks
2. Complete daily tasks to grow streaks
3. Beat your longest streak record
4. Get motivation from daily progress!

### **Personalize Settings** (5 minutes)
1. Profile → "My Goals" - set step & sleep targets
2. Profile → "Preferences" - toggle notifications, dark mode
3. Done! Settings sync across app
